Follow these steps to install Python, set up Jupyter Notebook, get your Gemini API key and run Ai Agent
1) follow link below and download python 3.11.0
https://www.python.org/downloads/release/python-3110/

2) install Jupyter and libraries  
pip install jupyterlab
pip install pandas sqlalchemy mysql-connector-python langchain langchain-community langchain-google-genai

3) start jupyter and run .ipynb file
jupyter notebook

4) paste code from Ai_Agent.py file and change values

5) get api key from https://aistudio.google.com/ and insert it into ai_agent

6) change value of user_question and run code
